# Tech Stack

## Frontend
- **React** with **TypeScript**
- **Tailwind CSS** — utility-first styling
- **React Router** — client-side routing

## Backend
- **Node.js** — server runtime
- **Express** — HTTP server and API routing

## Database
- **PostgreSQL** — primary relational database
- **Prisma** — ORM for type-safe database access

## Authentication
- **Database sessions** — server-side sessions stored in PostgreSQL

## AI
- **Anthropic Claude API** — ticket classification, summarization, and reply generation

## Email
- **Resend** — inbound email parsing (ticket ingestion) and outbound reply sending
